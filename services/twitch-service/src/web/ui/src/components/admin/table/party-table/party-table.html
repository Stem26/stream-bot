Партия
ВКЛ
Ограничение
ВКЛ
Например: !партия или !суккубы. В !команды подтягивается автоматически.
Например: Партия выдала или Суккубы выдали. Далее подставится «N элемент для @user».

Пример: триггер !партия, текст «партия выдали» → «партия выдали 3 дракоша‑сосед и 2 хомяко‑адвоката для @user». Раз в сутки на пользователя.

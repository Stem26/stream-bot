Управление ботом и командами

На главную

Инфо

!ссылки — редактировать общий текст, который отправляется при этой команде

!команды (или !команда, !commands) — выводит список всех включённых команд из таблицы ниже (триггеры и алиасы)

Команды для стримера и модераторов:

!title <название> — смена названия трансляции

!game <категория> — смена категории/игры (например: !game Just Chatting)

!tags [теги] — управление тегами стрима: !tags — показать, !tags tag1, tag2 — задать

Рейд — при входящем рейде бот отправит сообщение в чат и сделает шатаут. Плейсхолдеры: {from}, {viewers}

Друзья-стримеры

Как только ник из списка напишет в чат, бот автоматически сделает /shoutout.

Ротация команд

Бот по очереди шлёт объявления из команд с флагом «В ротации».

фываыфва команду

Команда для вызова в чате (например: !discord)
Дополнительные варианты команды через запятую
0 / 500
Текст, который бот отправит в чат. Макс. 500 символов в одном сообщении. Время: {time:Europe/Moscow}, {time:Asia/Tokyo} — любой IANA часовой пояс.
Объявления - яркие цветные, но с лимитами. Сообщения - обычные, можно чаще
Для сообщений: 0 — без кулдауна. Для объявлений: минимум 5 секунд. Для объявлений кулдаун должен быть не меньше 5 секунд.
